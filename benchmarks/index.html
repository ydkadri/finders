About These Benchmarks

Benchmarks test 6 scenarios: 3 repository sizes (small ~100 files, medium ~1K files, large ~10K files) × 2 search patterns (common pattern found in ~50% of files, rare pattern in 1 file).

All tools search for patterns in *.rs files using equivalent commands:

  • find + grep: Traditional Unix pipeline
  • ripgrep: Modern fast searcher
  • finder: This project's implementation
